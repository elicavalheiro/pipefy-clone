# pipefy-clone

This project was created to practice React and Drag n' drop function which is a very common function in applications nowdays. It was created following the video provided by Rocketseat: [Recriando Pipefy do zero (com drag n' drop) | Diego Fernandes](https://www.youtube.com/watch?v=awRtgpRsdTQ&list=PLscrnfGnuaklV7T3lXHGcnXj2Co0WgPf3&index=8&t=1s)

## Built With

* [React](https://www.google.com/search?q=react&oq=react&aqs=chrome.0.69i59j35i39l2j0j69i60l4.1916j0j7&sourceid=chrome&ie=UTF-8) - The front-end lib
* [styled-components](https://styled-components.com/) - To make the application looks good
* [react-icons](https://react-icons.github.io/react-icons/) - To add some beautiful icons :D
* [immer](https://react-icons.github.io/react-icons/)
